## Color Tokens

| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `background` | stone-50 | stone-900 | Page background |
| `foreground` | stone-900 | stone-50 | Primary text |
| `primary` | stone-900 | stone-200 | Buttons, links |
| `muted` | stone-100 | stone-700 | Secondary backgrounds |
| `border` | stone-200 | stone-600 | Borders, dividers |
